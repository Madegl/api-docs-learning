# Data Dictionary: Order Management System
  
**СУБД:** PostgreSQL

**Проект:** Orders

**Дата:** 24.05.2026

  
---

## 1. partners (Партнёры)
| Поле | Тип | Описание |
|------|-----|----------|
| `id` | INTEGER | Уникальный ID |
| `name` | TEXT | Название компании |
| `contact_email`| TEXT | Email для связи |
| `contract_date`| DATE | Дата начала договора |

## 2. status_dict (Статусы)
| Поле             | Тип     | Описание                  |
| ---------------- | ------- | ------------------------- |
| `status_code`    | TEXT    | Код (NEW, PAID...)        |
| `status_name`    | TEXT    | Название статуса          |
| `is_final`       | INTEGER | 0 - в процессе, 1 - финал |
| `status_type_id` | INTEGER | Id статуса заказа (FK)    |

## 3. status_type (Типы статусов)
| Поле          | Тип         | Описание                                |
| ------------- | ----------- | --------------------------------------- |
| `id`          | INTEGER     | Уникальный ID                           |
| `code`        | VARCHAR(30) | Код типа статуса                        |
| `name`        | VARCHAR(50) | Наименование типа статуса               |
| `description` | TEXT        | Описание                                |
| `is_active`   | BOOLEAN     | 0 - тип статуса не активен, 1 - активен |

## 4. orders (Заказы)
| Поле               | Тип      | Описание                                                              |
| ------------------ | -------- | --------------------------------------------------------------------- |
| `id`               | INTEGER  | Уникальный ID                                                         |
| `order_number`     | TEXT     | Номер заказа (ORD-...)                                                |
| `partner_id`       | INTEGER  | Ссылка на партнёра (FK)                                               |
| `created_at`       | DATETIME | Дата создания заказа                                                  |
| `delivery_address` | TEXT     | Адрес доставки                                                        |
| `delivery_method`  | TEXT     | Способ доставки                                                       |
| `total_amount`     | DECIMAL  | Итоговая сумма (кэш)                                                  |
| `status`           | INTEGER  | Текущий статус (FK)                                                   |
| `items_json`       | JSONB    | Информация о заказе. Содержит позиции товаров, количество, стоимость. |

## 5. order_status_history (История статусов)
| Поле          | Тип         | Описание             |
| ------------- | ----------- | -------------------- |
| `id`          | INTEGER     | Уникальный ID        |
| `order_id`    | INTEGER     | Ссылка на заказ (FK) |
| `status_code` | VARCHAR(30) | Код статуса (FK)     |
| `changed_at`  | TIMESTAMP   | Время изменения      |
| `changed_by`  | TEXT        | Кто изменил          |

## 6. partner_interaction_log (Взаимодействия)
| Поле               | Тип      | Описание                       |
| ------------------ | -------- | ------------------------------ |
| `id`               | INTEGER  | Уникальный ID                  |
| `partner_id`       | INTEGER  | Ссылка на партнёра (FK)        |
| `order_id`         | INTEGER  | Ссылка на заказ (FK, nullable) |
| `interaction_date` | DATETIME | Дата контакта                  |
| `interaction_type` | TEXT     | Тип (call, email...)           |
| `notes`            | TEXT     | Заметки                        |

## 7. returns

| Поле       | Тип         | Описание                                                                       |
| ---------- | ----------- | ------------------------------------------------------------------------------ |
| id         | INTEGER     | Уникальный ID                                                                  |
| order_id   | INTEGER     | Ссылка на заказ (FK)                                                           |
| items_json | JSONB       | Информация о возврате заказа. Содержит позиции товаров, количество, стоимость. |
| status     | VARCHAR(30) | Текущий статус (FK)                                                            |
| created_at | DATETIME    | Дата и время создания возврата                                                 |
| updated_at | DATETIME    | Дата и время изменения возврата                                                |


## 8. returns_status_history

| Поле        | Тип         | Описание                       |
| ----------- | ----------- | ------------------------------ |
| id          | INTEGER     | Уникальный ID                  |
| return_id   | INTEGER     | Ссылка на возврат (FK)         |
| status_code | VARCHAR(30) | Код статуса возрата (FK)       |
| changed_at  | DATETIME    | Дата и время создания возврата |
| changed_by  | TEXT        | Кем был создан возврат         |
| comment     | TEXT        | Примечание                     |
